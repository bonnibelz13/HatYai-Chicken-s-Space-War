ชื่อโครงงาน	:	HatYai Chicken’s Space War (สงครามอวกาศของไก่ทอดหาดใหญ่)

เป็นเกมยิงยานอวกาศของฝ่ายศัตรูผู้เล่นจะเล่นเป็นตัวเอกมาเฟียไก่หาดใหญ่ที่ต้องควบคุมยานอวกาศที่ต้องยิงกำจัดศัตรูคือสัตว์ปีกหลากหลายสายพันธุ์ เช่น เป็ด ห่าน ระหว่างที่ยิงผู้เล่นจะต้องหลบฝ่ายศัตรูที่ยิงโต้ตอบกลับมาด้วย หากผู้เล่นโดนยิงจะเสียพลังชีวิตไป 1 ชีวิต หากพลังชีวิตหมด จะจบเกมทันที

ความเป็นมา      	:
เนื่องจากสมาชิกท่านหนึ่งมีความชื่นชอบมาเฟียและไก่ทอดหาดใหญ่ จึงอยากสร้างโปรเจคเกมการต่อสู้ของมาเฟียไก่ทอดหาดใหญ่ด้วยภาษา Python ที่ได้เรียนจากวิชา PSCP ขึ้นมา รวมถึงความชอบในเกมแนว Retro สมัยก่อน โดยมีต้นแบบมาจากเกม Space Invaders(スペースインベーダ) วิดีโอเกมอาเขตแนวเกมยิงแก้ปัญหา 2 มิติ ของประเทศญี่ปุ่น ซึ่งเป็นเกมยิงปืนอวกาศในตำนานที่เก่าแก่ที่สุด จึงได้ผสมเนื้อเรื่องของมาเฟียไก่ทอด และแนวเกมเข้าด้วยกัน เกิดเป็นโปรเจคเกมสงครามอวกาศของไก่ทอดหาดใหญ่ขึ้นมา

การป้อนข้อมูล (Input)	:   กดปุ่ม A,  D และ SpaceBar เพื่อเคลื่อนที่ไปทางซ้าย ขวา และยิงกระสุนตามลำดับ
ผลลัพธ์ (Output) 	:   ตัวละครเคลื่อนที่ตามการกดของแป้นคีย์บอร์ดปุ่ม A, D และ SpaceBar ยิงกระสุน

การแบ่งงาน	:
มีการใช้ฟังก์ชัน แบ่งออกเป็นส่วน ๆ ได้แก่ ผู้เล่น, ศัตรู, กระสุน, ส่วนหน้าสกรีน, ส่วนแสดงคะแนน และส่วนของตัวเกม คือ การเคลื่อนที่ซ้าย-ขวา, การยิงกระสุน, การตายของฝ่ายศัตรู, การเสียค่าพลังชีวิตของผู้เล่น


ประโยชน์	:
ฝึกทักษะความคิดในการแก้ปัญหาได้ดีขึ้น เกมสงครามอวกาศของไก่ทอดหาดใหญ่ใช้การแก้ปัญหาเฉพาะหน้า ในการตัดสินใจคิดเพื่อหลบหลีกกระสุนของศัตรู และต้องคิดในเวลาที่ยิงโต้ตอบกลับไปด้วย ช่วยส่งเสริมความสามารถในการควบคุมการทำงานของกล้ามเนื้อมือและตาให้ทำงานประสานกันได้ดี
